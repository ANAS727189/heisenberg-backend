# Heisenberg Backend

A backend service running Kestra workflow orchestration platform in a containerized environment. This service provides workflow automation capabilities and exposes its API endpoint to be consumed by frontend applications.

## Overview

This repository contains the backend infrastructure for the Heisenberg project, leveraging [Kestra](https://kestra.io/) - an open-source orchestration and scheduling platform for building, running, scheduling, and monitoring complex pipelines.
